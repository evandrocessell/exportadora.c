# Calculadora de Tarifa de Exportação (C)

## Sobre o Projeto

Este programa em C calcula o **custo total de uma exportação**, somando o valor da mercadoria com a tarifa cobrada pelo país de destino.

**As tarifas aplicadas são:**

País de Destino | Código | Tarifa (Imposto) 
:--- | :--- | :--- 
Alemanha | **A** | 65% (0.65) 
Espanha | **E** | 47% (0.47) 
Itália | **I** | 45% (0.45)
